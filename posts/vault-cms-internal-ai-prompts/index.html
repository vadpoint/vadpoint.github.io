Extract Tasks

On the current date {{date}}, extract all to-do items from {{file:content}}, focusing on: {{selection}}

Please output the results strictly in the Obsidian Tasks plugin format, with each task on a separate line:

  • Task description ⏫/🔼/🔽/⏬ Priority
  • Task description 📅 YYYY-MM-DD Due date
  • Task description ⏰ YYYY-MM-DD HH:mm Reminder time
  • Task description 🛫 YYYY-MM-DD Start date
  • Task description 🔁 every day/week/month Recurrence
  • Task description #tag #project_name

Extraction rules:

  1. If the original text mentions “urgent/immediately/today,” add ⏫ (highest priority)
  2. If it mentions “tomorrow/this week/as soon as possible,” add 🔼 (high priority)
  3. If a specific date is mentioned, convert it to 📅 YYYY-MM-DD format
  4. If a time is mentioned, convert it to ⏰ YYYY-MM-DD HH:mm format
  5. If a recurring task is mentioned, add 🔁 every week/month
  6. Add appropriate tags to each task (e.g., #work #meeting #follow-up)

Sample output:

  • Complete project report ⏫ 📅 2026-04-25 #Work
  • Weekly team meeting 🔁 every week on Monday ⏰ 09:00 #Meeting
  • Follow up on client requirements 🔼 📅 2026-04-23 #FollowUp
