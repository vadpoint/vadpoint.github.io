SDD Journey: Фікс Node.js версії та оточення для Astro Pure

Наступним кроком у стабілізації нашого пайплайну стало вирішення проблем з оточенням, які виникли під час роботи Astro 6 та специфічних скриптів нашої теми.

Аналіз проблем

  1. Node.js Compatibility: Astro v6 вимагає Node.js версії >=22.12.0. Хоча ми використовуємо Bun як основний рантайм для інсталяції та запуску скриптів, Astro (та пакет astro-check) все одно перевіряють версію Node, встановлену в системі. Runner ubuntu-latest за замовчуванням мав версію 20, що призвело до зупинки білду.
  2. BUN_LINK_PKG: Скрипт check.mjs, який є частиною теми astro-pure, вимагає явного підтвердження через змінну оточення BUN_LINK_PKG=true. Без неї він пропускає важливі етапи перевірки, що може заблокувати подальший процес.

Що було зроблено:

  • deploy.yml: Додано крок actions/setup-node@v4 з явною версією 24. Це гарантує відповідність вимогам Astro v6.
  • Environment Variables: У глобальні налаштування джоби додано BUN_LINK_PKG: true.

Тепер пайплайн має все необхідне: правильний Node.js для внутрішніх перевірок Astro та Bun для швидкої роботи із залежностями.
